# Prompt Engineering for Developers

This repository provides a practical and engineering-oriented
training on prompt design for large language models.

The focus is on:

- structured prompt design
- reusable prompt patterns
- systematic evaluation
- and real-world system integration.

---

## Why this repository?

Most prompt resources focus on writing better prompts.

This repository focuses on designing prompt systems.

---

- 05-prompt-engineering-with-api  

---

## Target audience

- software engineers
- data scientists
- AI practitioners
- product and business analysts

---

## How to use this repository

Read the sections in order.

Apply the structures to your own projects.

Treat prompts as system components.
